┌─────────────────────────────────────────────────────────────────────────┐
│ 💡 KEY INSIGHTS                                                          │
└─────────────────────────────────────────────────────────────────────────┘

  1. PnP (Angular SPA)
     • Products loaded via JavaScript API calls
     • Must use Selenium + network monitoring
     • Slower but gets more products per page
     • No anti-bot protection
  
  2. Shoprite (React SSR)
     • Products already in HTML
     • Simple requests + BeautifulSoup works
     • Much faster but fewer products per page
     • Blocks Selenium (anti-bot active)
  
  3. Different retailers need different approaches!
