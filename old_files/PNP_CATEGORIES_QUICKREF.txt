USAGE EXAMPLES
───────────────────────────────────────────────────────────────────────────

Single Category:
  scraper = PnPSmartScraper(category='beverages')
  products = scraper.scrape(max_pages=3)

Multiple Categories:
  for cat in ['snacks', 'beverages', 'dairy']:
      scraper = PnPSmartScraper(category=cat)
      products = scraper.scrape(max_pages=1)
      scraper.save_json()

With Limits:
  scraper = PnPSmartScraper(category='liquor')
  products = scraper.scrape(max_pages=5, max_products=100)


FILE NAMING
───────────────────────────────────────────────────────────────────────────

Auto-naming by category:
  category='snacks'    → pnp_snacks_products.json
  category='beverages' → pnp_beverages_products.json
  category='liquor'    → pnp_liquor_products.json

Custom naming:
  scraper.save_json('my_custom_name.json')
  scraper.save_csv('my_custom_name.csv')
